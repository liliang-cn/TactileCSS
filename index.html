Tactile CSS

Neumorphic UI with physical material themes

Soft, Tactile, Beautiful

A minimal CSS library providing neumorphic shadows (outer, inner, clay) and physical material themes. Just add classes to any element.

GitHub Get Started

Three Shadow Styles

Outer (raised), Inner (sculpted), and Clay (fluffy) shadow styles for different visual effects.

Six Themes

Classic, Mono, Iron, Paper, Terminal, and Warm themes with carefully crafted color palettes.

Zero Dependencies

Pure CSS, no JavaScript required. Works with any framework or vanilla HTML.

Quick Start

<!-- Add to your HTML -->
<link rel="stylesheet" href="tactile.css">
<body class="tactile">

<!-- Shadow styles -->
<div class="tactile-outer">Raised</div>
<div class="tactile-inner">Inset</div>
<div class="tactile-clay">Fluffy</div>

<!-- Size variants -->
<div class="tactile-outer tactile-sm">Small</div>
<div class="tactile-outer tactile-lg">Large</div>

<!-- Buttons -->
<button class="tactile-button">Default</button>
<button class="tactile-button-primary">Primary</button>
<button class="tactile-button-clay">Clay</button>

<!-- Theme switch -->
<html data-theme="iron">
